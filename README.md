# Lyfe Run

Initially I need a Landing page, where a platform called Lyfe Run will be offered to Running Coaches that want to create a coaching business that is virtual but also done in person. The platform allows you to:

a) Configure your coaching practice, your additional trainers and your current clients (runners) using a backoffice interface. The current clients will have the option to pay through credit cards, bank payments, instant transfer (Brazilian "PIX" standard).

b) Provides you with a personalized landing page where you can offer your coaching practice to new users. Offers onboarding and payment configuration for this new users.

c) The coach will be able to provide customized daily plans using a weekly window. He will also be able to update the plan when needed and provide new running plans every monday for instance.

d) The Coach will have access to an AI plan to start with.

e) The coach will be able to configure his "catalog" of running plans that will be filtered and chosen via parameters such as the client's goals, running experience, actual running tests to validate self supplied information.

f) The platform will provide "Garmin/Apple Watch, Whoop like" interfaces where the user will be able to receive his plan directly on the device and also automatically record it.

g) If the runner does not have a telemetry device such as the ones mentioned above, he will be able to record the training directly on the app using GPS.

h) The Coach will have access to the actual execution information (planned x actual), heart rate frequency during each session, sleep quality/score, and actual running speed for each segment. With that information and suggest AI feedback will be generated, the coach will either accept it and send, edit or provide a totally custom message. In time this will offload most of the customer relationship work from the coach.

i) After every run the platform will update your VO2 estimation, your predicted performance for each distance (5km, 10km, half and full marathon).

j) The Runner will be able to provide his planned races from a centralized catalog.

k) The runner will be able to buy the tickets for the Races when available.

l) There will be a seamless integration with a payment gateway.

m) There will be two plans, the coach will pay 200 BRL per month for the basic plan or 3% of the monthly revenue for the gold plan with access to the AI features.
